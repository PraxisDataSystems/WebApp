# Propstream Export Manager

Minimal Next.js app for managing Propstream property exports.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deployment

This app deploys to Vercel. Push to main branch to trigger auto-deploy.

## Notes

- This is just the UI shell
- Agent automation runs separately on the OpenClaw server
- Expand this app with Claude Code as needed
